# VS Playwright Course

A Playwright test course repository with workshop exercises and example tests.

## Project

- Name: vs-playwright-course
- Version: 1.0.0
- Author: Vasyl Shpak
- License: MIT

## Contents

- `playwright.config.ts` - Playwright test configuration
- `tests/` - Workshop exercise tests organized by workshop number
- `tests-examples/` - Example tests for demonstration

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run Playwright tests:
   ```bash
   npx playwright test
   ```

## Notes

This repository uses `@playwright/test` for browser automation and testing. Add or update scripts in `package.json` as needed for your workflow.
