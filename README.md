## Sobre o Projeto

Este repositório reúne os exercícios, exemplos e práticas desenvolvidos durante um curso de **Automação de Testes com Playwright**.

O objetivo foi aprender os principais conceitos da ferramenta, incluindo criação de testes automatizados, interação com elementos da interface, validações, configurações do framework e boas práticas para testes End-to-End (E2E).

---

## O que foi praticado

*  Criação de testes automatizados com Playwright
*  Interação com elementos da página
*  Validações e assertions
*  Automação de fluxos web
*  Organização de suítes de testes
*  Configuração do Playwright
*  Execução de testes em diferentes navegadores
*  Geração de relatórios de execução

## Para executar o projeto:

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/vs-playwright-course.git
```

### Acesse a pasta do projeto

```bash
cd vs-playwright-course
```

### Instale as dependências

```bash
npm install
```

### Execute todos os testes

```bash
npx playwright test
```

### Executar os testes em modo visual

```bash
npx playwright test --ui
```

### Abrir o relatório após a execução

```bash
npx playwright show-report
```
